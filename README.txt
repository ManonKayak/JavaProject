Bienvenue sur notre projet de gestion d'une base de donnée de programmeurs !

Pour pouvoir lancer notre projet, vous aurez besoin au préalable :

    1. D'installer Docker Desktop - https://www.docker.com/products/docker-desktop/.
    2. Une fois installé, vérifiez que l’icône Docker affiche "running".
    3. Si vous utilisez IntelliJ, installer le plugin Docker.
    4. Assurez-vous qu'il est bien installé en allant dans "Settings->Build, Execution, Deployment->Docker".
    5. Lancez le fichier docker-compose.yml.
    6. Enfin, lancez le fichier Main.java et laissez-vous guider !
